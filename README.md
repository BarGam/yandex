# Клуб четырёх коней

Лендинг «Международный васюкинский турнир» — адаптивная вёрстка (375 / 1366 / 1920).

## Запуск локально

```bash
python3 -m http.server 8080
```

Открыть http://127.0.0.1:8080/

## Структура

- `index.html` — разметка
- `css/` — стили
- `js/` — слайдеры, бегущая строка
- `img/` — изображения (реальные файлы, не симлинки)
- `fonts/` — Golos Text, Merriweather

## Деплой

Статический сайт: Netlify, GitHub Pages или любой хостинг с корнем в папке проекта.
